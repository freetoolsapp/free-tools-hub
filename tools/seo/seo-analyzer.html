What Does SEO Analyzer Check?

Title Tag

Checks if title exists, its length (ideal: 50–60 chars), and if your keyword is included.

Meta Description

Verifies meta description presence and length (ideal: 150–160 characters).

Heading Structure

Checks H1–H6 tags: one H1 required, proper hierarchy, keyword in H1.

Image Alt Texts

Finds all images and checks which ones are missing alt attributes.

Links Analysis

Counts internal links, external links, and checks for nofollow attributes.

Keyword Density

Shows top keywords and their frequency. Ideal density: 1–2% for target keyword.
